# CandyWorld

Unityで制作した一人称視点のお菓子の世界が舞台となったダンジョンゲームです。

## 推奨環境
* **OS**: Windows 11 以上（推奨）
* **動作保証**: 本ソフトウェアは個人制作のプロトタイプであり、すべての環境での動作を保証するものではありません。あらかじめご了承ください。

## 開発環境
* **Unity**: 2021.3.45f1
* **レンダリングパイプライン**: URP (Universal Render Pipeline)
* **開発ツール**: Visual Studio 2022 / Visual Studio 2019
* **その他使用ソフト**: Blender / Scaniverse

## 遊び方（Windows版）
1. [リリースページ](https://github.com/S40217kot/CandyWorld-Windows/releases) にアクセスします。
2. 最新の Zip ファイルをダウンロードして解凍します。
3. フォルダの中にある `My project.exe` を実行してください。

## 操作方法
* **移動**: WASD
* **視点移動**: マウス
* **ジャンプ**: Space
* **終了**: Alt + F4

## 使用素材クレジット
* **グラフィック**：いらすとや 様
* **BGM**：魔王魂 様
* **システム**：Unity Starter Assets - First Person Character Controller
